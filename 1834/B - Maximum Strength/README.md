Help him find the maximum possible strength of the weapon he can obtain.</p><p>An integer $$$C = \overline{c_{1}c_{2} \ldots c_{k}}$$$ is defined as an integer obtained by sequentially writing the digits $$$c_1, c_2, \ldots, c_k$$$ from left to right, i.e. $$$10^{k-1} \cdot c_1 + 10^{k-2} \cdot c_2 + \ldots + c_k$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases.
